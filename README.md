## Example app

An app that uses all features of `ngx-forms-typed`.

Build and run using

`ngx ng build forms && ng serve`

OR using 2 separate terminal windows
`ngx ng build forms -- --watch` in a terminal and `ngx ng serve` in the second terminal window
